## Створені бат-файли

### 1. **run.bat** - Звичайний запуск
- Перевіряє встановлення Python
- Перевіряє/встановлює залежності
- Перевіряє доступність Ollama
- Запускає програму з повною перевіркою

**Використання:**
```
run.bat
```

### 2. **run-quick.bat** - Швидкий запуск
- Пропускає перевірку Ollama
- Корисно якщо Ollama ще не запущено

**Використання:**
```
run-quick.bat
```

### 3. **run-unsafe.bat** - Режим повного доступу
- Повний доступ до файлової системи
- Дозволяє запуск програм
- Без перевірок безпеки

**Використання:**
```
run-unsafe.bat
```

### 4. **run-streaming.bat** - Інтерактивний режим (Streaming) ✨
- НОВА ФУНКЦІЯ!
- Відповіді з'являються в реальному часі, як у Gemini CLI або ChatGPT
- Текст друкується поступово, не треба чекати кінця генерації
- Ідеально для довгих відповідей

**Використання:**
```
run-streaming.bat
```

### 5. **Medfarl_Menu.bat** - Меню запуску
- Інтерактивне меню з усіма опціями
- Звичайний запуск
- Швидкий запуск
- Unsafe режим
- **Опція 16: Streaming режим** (НОВЕ!)
- Перевірка здоров'я
- Список моделей
- Бенчмарк

**Використання:**
```
Medfarl_Menu.bat
```

## Вимоги

1. **Python 3.10+** - https://python.org
2. **Ollama** - https://ollama.ai (опціонально)
3. **Модель** - будь-яка з підтримкою tool calling (llama3.2, qwen2.5, mistral-nemo)

## Встановлення

```bash
# Встановити залежності
pip install -r requirements.txt

# Запустити Ollama та завантажити модель
ollama pull qwen3.5:9b
```

## Команди запуску

```bash
# Звичайний запуск
python main.py

# Без перевірки Ollama
python main.py --skip-healthcheck

# З повним доступом
python main.py --unsafe-full-access --skip-healthcheck

# ІНТЕРАКТИВНИЙ РЕЖИМ (Streaming)
python main.py --stream
python main.py --stream --model qwen3.5:9b

# Перевірка системи
python main.py --healthcheck

# Список моделей
python main.py --list-models

# Бенчмарк
python main.py --benchmark-models llama3.2 qwen3.5:9b
```

## Вирішення проблем

### Помилка "Cannot reach Ollama"
- Переконайтесь що Ollama запущено
- Перевірте URL: http://localhost:11434
- Використовуйте `--skip-healthcheck` для запуску без перевірки

### Помилка імпорту модулів
```bash
pip install -r requirements.txt
```

### Повільна відповідь
- Збільште таймаут: `python main.py --timeout 240`
- Використовуйте легшу модель

## ✨ НОВА ФУНКЦІЯ: Streaming Режим

### Що таке streaming?
Текст відповіді з'являється **в реальному часі**, токен за токеном, прямо як у:
- Gemini CLI
- ChatGPT
- Claude

### Як це виглядає:

#### Звичайний режим (OLD):
```
medfarl> привіт
[чекаємо 15 секунд...]
[вся відповідь з'являється одразу]
```

#### Streaming режим (NEW):
```
medfarl> привіт
Привіт! ← з'являється одразу
Що саме перевірити: ← друк в реальному часі
- загальний стан системи
- процеси, диски, мережу чи логи?
```

## 🎯 Технічні деталі streaming

### Архітектура:

```
User Input
    ↓
Agent (stream=True)
    ↓
LLMClient._stream_response()
    ↓
httpx.stream("POST", ...)
    ↓
For each SSE "data:" line:
    - Parse JSON chunk
    - Extract delta.content
    - print(content, end="", flush=True) ← миттєвий друк
    - Accumulate full_content
    ↓
Return complete response
```

### Protocol:
- **Server-Sent Events (SSE)**
- Format: `data: {json}\n\n`
- End marker: `data: [DONE]\n\n`
